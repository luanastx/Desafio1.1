TECNOLOGIAS

Tecnologias utilizadas

HTML

HTML é a linguagem padrão para criar páginas da web. Ele descreve a estrutura de uma página web utilizando uma série de elementos e tags.

Site oficial: W3C HTML

    Prós:

  • Facilidade de Uso
  • Suporte Amplo
  • Base para o Desenvolvimento Web

    Limitações:

  • Não é uma Linguagem de Programação
  • Dependência de Outras Tecnologias
  • Estático

    Grandes projetos:

  • Google
  • Wikipedia
  • Facebook
  • YouTube
  • Amazon
  • Twitter
CSS

CSS é a linguagem usada para descrever a apresentação de documentos HTML. Ele controla o layout, cores, fontes e o design geral da página.

Site oficial: W3C CSS

    Prós:

  • Separação de Conteúdo e Estilo
  • Reutilização
  • Customização

    Limitações:

  • Compatibilidade entre Navegadores
  • Complexidade
  • Dependência de HTML

    Grandes projetos:

  • YouTube
  • Twitter
  • Amazon
  • Google
  • Wikipedia
  • Facebook

JavaScript

JavaScript é uma linguagem de programação que permite criar conteúdo dinâmico e interativo em páginas web.

Site oficial: JavaScript

    Prós:

  • Interatividade
  • Versatilidade
  • Bibliotecas e Frameworks

    Limitações:

  • Desempenho
  • Segurança
  • Dependência de Navegador

    Grandes projetos:

  • Google
Java

Java é uma linguagem de programação de propósito geral, orientada a objetos e baseada em classes, projetada para ser executada em qualquer plataforma através da JVM (Java Virtual Machine)

Site oficial: Java

    Prós:

  • Portabilidade
  • Desempenho
  • Segurança

    Limitações:

  • Complexidade
  • Consumo de Memória

  • LinkedIn
